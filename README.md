This project is an Automated Resume Screening System built using Natural Language Processing (NLP) techniques. 
It helps in filtering and ranking resumes based on their relevance to a given job description.

The system performs text preprocessing such as tokenization, stop-word removal, and stemming to clean resume data. 
It then applies keyword extraction and cosine similarity to match resumes with job requirements.

This project reduces manual effort in recruitment and improves the efficiency of candidate shortlisting.
